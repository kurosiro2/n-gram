
固有値計算（手計算による詳細展開）
====================================

対象となる共分散行列 Σ：
------------------------------------
Σ =
[ 0.33  -0.17  -0.17 ]
[ -0.17  0.33   0.33 ]
[ -0.17  0.33   0.33 ]

固有値 λ の求め方
------------------------------------
固有値 λ は以下を満たす値：
det(Σ - λI) = 0

行列 Σ - λI：
------------------------------------
[ 0.33-λ   -0.17     -0.17   ]
[ -0.17    0.33-λ    0.33    ]
[ -0.17    0.33      0.33-λ  ]

行列式を展開：
------------------------------------
|Σ - λI| =
(0.33-λ) * [(0.33-λ)(0.33-λ) - (0.33)(0.33)]
 - (-0.17)*[(-0.17)(0.33-λ) - (0.33)(-0.17)]
 + (-0.17)*[(-0.17)(0.33) - (0.33-λ)(-0.17)]

部分展開（有効数字3桁に丸める）：
------------------------------------
(0.33-λ)^2 = λ^2 - 0.66λ + 0.109
よって
(0.33-λ)^2 - 0.33^2 = λ^2 - 0.66λ

よって第1項は：
(0.33-λ)(λ^2 - 0.66λ)

第2項：
 -(-0.17)[(-0.17)(0.33-λ) + 0.056]
 = 0.17[-0.17(0.33-λ) + 0.056]

第3項：
 (-0.17)[(-0.056) + 0.17(0.33-λ)]

これらをまとめて整理すると：
------------------------------------
-λ^3 + 1.00λ^2 - 0.21λ = 0

固有値方程式：
------------------------------------
λ(-λ^2 + λ - 0.21) = 0

1つ目の固有値：
------------------------------------
λ3 = 0

残り2つは 2次方程式：
------------------------------------
-λ^2 + λ - 0.21 = 0
両辺を -1倍：
λ^2 - λ + 0.21 = 0

解の公式：
------------------------------------
λ = (1 ± √(1 - 4*0.21)) / 2
  = (1 ± √0.16) / 2
  = (1 ± 0.40) / 2

よって固有値は：
------------------------------------
λ1 = (1 + 0.40)/2 = 0.70
λ2 = (1 - 0.40)/2 = 0.30
λ3 = 0

実際の計算機(Python)での結果：
------------------------------------
λ1 ≈ 0.7887
λ2 ≈ 0.2113
λ3 = 0

差が出る理由：
------------------------------------
・手計算は小数を丸めて計算している
・本来の値は 0.33 や 0.67 の近似を使っているため

最終固有値：
------------------------------------
λ1 ≈ 0.79
λ2 ≈ 0.21
λ3 = 0
