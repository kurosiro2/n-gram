PCA 手計算メモ（3人 A,B,C の簡単な例）

============================================================
0. 前提
============================================================

5 次元ベクトル（SE->SN, SN->WR, D->D, E->N, LD->SE）のうち、
今回の例では E->N, LD->SE は全員 0 なので、実質 3 次元に簡略化して考える。

次の 3 つの特徴だけを使う：

  x1 = SE->SN
  x2 = SN->WR
  x3 = D->D

与えられたデータ：

  A さん: SE->SN=1, SN->WR=0, D->D=0
  B さん: SE->SN=0, SN->WR=0, D->D=0
  C さん: SE->SN=0, SN->WR=1, D->D=1

ベクトル表現：

  A = (1, 0, 0)
  B = (0, 0, 0)
  C = (0, 1, 1)

これを PCA にかけて、固有値・固有ベクトル（PC1, PC2, PC3）を求める。

============================================================
1. 平均ベクトル（mean）の計算
============================================================

各次元ごとに平均をとる：

  mean_x1 = (1 + 0 + 0) / 3 = 1/3 ≒ 0.33
  mean_x2 = (0 + 0 + 1) / 3 = 1/3 ≒ 0.33
  mean_x3 = (0 + 0 + 1) / 3 = 1/3 ≒ 0.33

よって平均ベクトル μ は：

  μ = (0.33, 0.33, 0.33)  （有効数字 2 桁程度）

============================================================
2. データの中心化（mean を引く）
============================================================

各人のベクトルから平均 μ を引く。

------------------------------------------------------------
A さん：
------------------------------------------------------------

  A = (1, 0, 0)

  A - μ = (1 - 0.33, 0 - 0.33, 0 - 0.33)
        = (0.67, -0.33, -0.33)

------------------------------------------------------------
B さん：
------------------------------------------------------------

  B = (0, 0, 0)

  B - μ = (0 - 0.33, 0 - 0.33, 0 - 0.33)
        = (-0.33, -0.33, -0.33)

------------------------------------------------------------
C さん：
------------------------------------------------------------

  C = (0, 1, 1)

  C - μ = (0 - 0.33, 1 - 0.33, 1 - 0.33)
        = (-0.33, 0.67, 0.67)

------------------------------------------------------------
中心化後の行列 Xc
------------------------------------------------------------

行ごとに A, B, C を並べると、

  Xc =
    [  0.67  -0.33  -0.33 ]   ← A_c
    [ -0.33  -0.33  -0.33 ]   ← B_c
    [ -0.33   0.67   0.67 ]   ← C_c

============================================================
3. 共分散行列 Σ の計算
============================================================

共分散行列の定義（サンプル共分散）：

  Σ = (1 / (n - 1)) * Xc^T * Xc

ここで n = 3（A,B,C の3人）なので、

  係数 = 1 / (3 - 1) = 1 / 2 = 0.5

まずは Xc^T * Xc の各要素（内積）を計算し、
最後に 0.5 をかければよい。

------------------------------------------------------------
3.1 各次元を列として抜き出す
------------------------------------------------------------

  x1 ベクトル = (0.67, -0.33, -0.33)
  x2 ベクトル = (-0.33, -0.33, 0.67)
  x3 ベクトル = (-0.33, -0.33, 0.67)

見てわかる通り、x2 と x3 は全く同じ動きをしている。

------------------------------------------------------------
3.2 Σ11 = Var(x1) の計算
------------------------------------------------------------

  Σ11 = (1 / (n - 1)) * Σ_i (x1_i)^2

  x1 の各成分の二乗：
    (0.67)^2 ≒ 0.4489 ≒ 0.45
    (-0.33)^2 ≒ 0.1089 ≒ 0.11
    (-0.33)^2 ≒ 0.11

  合計 ≒ 0.45 + 0.11 + 0.11 = 0.67

  Σ11 = 0.67 / 2 ≒ 0.33

------------------------------------------------------------
3.3 Σ22 = Var(x2) の計算
------------------------------------------------------------

  x2 = (-0.33, -0.33, 0.67)

  二乗：
    (-0.33)^2 = 0.11
    (-0.33)^2 = 0.11
    (0.67)^2  = 0.45

  合計 0.11 + 0.11 + 0.45 = 0.67

  Σ22 = 0.67 / 2 ≒ 0.33

------------------------------------------------------------
3.4 Σ33 = Var(x3) の計算
------------------------------------------------------------

  x3 も x2 と同じなので、

  Σ33 = 0.33

------------------------------------------------------------
3.5 Σ12 = Cov(x1, x2) の計算
------------------------------------------------------------

共分散：

  Σ12 = (1 / (n - 1)) * Σ_i x1_i * x2_i

  各項：

    i=1: x1_1 * x2_1 = 0.67 * (-0.33) ≒ -0.22
    i=2: x1_2 * x2_2 = (-0.33)*(-0.33) ≒ 0.11
    i=3: x1_3 * x2_3 = (-0.33)*(0.67) ≒ -0.22

  合計 ≒ -0.22 + 0.11 - 0.22 = -0.33

  Σ12 = -0.33 / 2 ≒ -0.17

共分散行列は対称なので Σ21 も同じ：

  Σ21 = Σ12 ≒ -0.17

------------------------------------------------------------
3.6 Σ13 = Cov(x1, x3)
------------------------------------------------------------

x3 = x2 と同じなので、

  Σ13 = Σ12 ≒ -0.17
  Σ31 = Σ13 ≒ -0.17

------------------------------------------------------------
3.7 Σ23 = Cov(x2, x3)
------------------------------------------------------------

x2 と x3 が完全に同じ動きをするので、共分散は分散と同じになる。

  Σ23 = Var(x2) = 0.33
  Σ32 = Σ23 = 0.33

------------------------------------------------------------
3.8 共分散行列 Σ の完成
------------------------------------------------------------

以上をまとめると、

  Σ =
    [  0.33  -0.17  -0.17 ]
    [ -0.17   0.33   0.33 ]
    [ -0.17   0.33   0.33 ]

============================================================
4. 固有値・固有ベクトルの計算
============================================================

PCA はこの共分散行列 Σ に対し、

  Σ w = λ w

を満たす固有値 λ と固有ベクトル w を求める。

------------------------------------------------------------
4.1 固有値 λ の概略
------------------------------------------------------------

この行列は x2, x3 が同じ動きをする特殊な形なので、
対称性から次のような固有値になることが分かる：

  λ1 ≒ 0.79
  λ2 ≒ 0.21
  λ3 = 0

※ λ3 = 0 になる理由：
  x2, x3 は常に一緒に動いており、その「差」方向にはまったく分散がないため。

------------------------------------------------------------
4.2 λ1 に対する固有ベクトル（PC1）の求め方
------------------------------------------------------------

λ1 ≒ 0.79 のとき、

  (Σ - λ1 I) w = 0

を満たす w = (w1, w2, w3) を探す。

式を書くと：

  (0.33 - 0.79) w1 + (-0.17)      w2 + (-0.17)      w3 = 0
  (-0.17)      w1 + (0.33 - 0.79) w2 + (0.33)       w3 = 0
  (-0.17)      w1 + (0.33)        w2 + (0.33 - 0.79)w3 = 0

  0.33 - 0.79 = -0.46 くらいなので、ざっくり：

  -0.46 w1 -0.17 w2 -0.17 w3 = 0      ...(1)
  -0.17 w1 -0.46 w2 +0.33 w3 = 0      ...(2)
  -0.17 w1 +0.33 w2 -0.46 w3 = 0      ...(3)

(2) と (3) を見ると、w2 と w3 がほぼ同じ役割をしていることが分かるため、
まず w2 = w3 と仮定してみる。

w2 = w3 = t とおくと、

(2) は：

  -0.17 w1 -0.46 t + 0.33 t = 0
  -0.17 w1 -0.13 t = 0
  0.17 w1 = -0.13 t
  w1 = - (0.13 / 0.17) t ≒ -0.76 t

また (1) に代入しても矛盾しないか確認する：

(1)
  -0.46 w1 -0.17 t -0.17 t = 0
  -0.46 w1 -0.34 t = 0
  -0.46 (-0.76 t) -0.34 t ≒ 0.35 t -0.34 t ≒ 0.01 t ≈ 0 （OK）

したがって比として、

  w1 : w2 : w3 ≒ -0.76 : 1 : 1

長さが1になるように正規化したものが、PC1 の固有ベクトル：

  PC1 ≒ (-0.46, 0.63, 0.63)

（※ 正規化のステップ：ベクトルの長さで割っている）

------------------------------------------------------------
4.3 λ2 に対する固有ベクトル（PC2）
------------------------------------------------------------

λ2 ≒ 0.21 に対して同様の計算をすると、
PC1 と直交する方向で、かつ分散が次に大きい方向として、

  PC2 ≒ (0.89, 0.33, 0.33)

となる（これも長さ1に正規化されている）。

これは「SE->SN に強く反応する軸」であり、
A さん（SE->SN が1）の寄与が大きくなる。

------------------------------------------------------------
4.4 λ3 = 0 に対する固有ベクトル（PC3）
------------------------------------------------------------

x2 と x3 が常に同時に増減するので、その「差」方向には一切分散がない。
したがって、x2, x3 の差を表すようなベクトル

  PC3 ≒ (0, -1, 1)

をとると、

  Σ PC3 = 0 * PC3

を満たす（これも正規化すると (0, -0.71, 0.71) くらい）。

しかし λ3=0 なので、この方向には情報がなく、
PCA では通常使われない。

============================================================
5. PCA 座標への射影（PC1・PC2）
============================================================

中心化後の各点 A_c, B_c, C_c を PC1, PC2 軸に射影する。

------------------------------------------------------------
5.1 A さんの射影
------------------------------------------------------------

  A_c = (0.67, -0.33, -0.33)

PC1 方向への射影：

  PC1 = (-0.46, 0.63, 0.63)

  PC1_A = A_c ・ PC1
         ≒ 0.67*(-0.46) + (-0.33)*0.63 + (-0.33)*0.63
         ≒ -0.31 - 0.21 - 0.21
         ≒ -0.73  （おおよそ）

PC2 方向への射影：

  PC2 = (0.89, 0.33, 0.33)

  PC2_A = A_c ・ PC2
         ≒ 0.67*0.89 + (-0.33)*0.33 + (-0.33)*0.33
         ≒ 0.60 - 0.11 - 0.11
         ≒ 0.38

従って A さんのおおよその PCA 座標は：

  A ≒ (PC1, PC2) = (-0.73, 0.38)

------------------------------------------------------------
5.2 B さんの射影
------------------------------------------------------------

  B_c = (-0.33, -0.33, -0.33)

  PC1_B ≒ (-0.33)*(-0.46) + (-0.33)*0.63 + (-0.33)*0.63
         ≒ 0.15 - 0.21 - 0.21
         ≒ -0.27

  PC2_B ≒ (-0.33)*0.89 + (-0.33)*0.33 + (-0.33)*0.33
         ≒ -0.29 - 0.11 - 0.11
         ≒ -0.51

  B ≒ (-0.27, -0.51)

------------------------------------------------------------
5.3 C さんの射影
------------------------------------------------------------

  C_c = (-0.33, 0.67, 0.67)

  PC1_C ≒ (-0.33)*(-0.46) + 0.67*0.63 + 0.67*0.63
         ≒ 0.15 + 0.42 + 0.42
         ≒ 0.99

  PC2_C ≒ (-0.33)*0.89 + 0.67*0.33 + 0.67*0.33
         ≒ -0.29 + 0.22 + 0.22
         ≒ 0.15

  C ≒ (0.99, 0.15)

============================================================
6. 解釈
============================================================

・PC1 は「C（SN->WR, D->D が強い） vs A（SE->SN が強い）」を
  一番強く引き離す軸になっている。
  → SN->WR, D->D の係数が正、SE->SN の係数が負。

・PC2 は「SE->SN が強い方向」の軸になっていて、
  A を上方向に持ち上げる役割がある。

・B は全ての特徴が小さいため、PC1, PC2 両方で中間〜やや下側の位置に来る。

このようにして、
「具体的な数値計算」として PCA の固有値・固有ベクトル・射影座標まで
全部手で追うことができる。
